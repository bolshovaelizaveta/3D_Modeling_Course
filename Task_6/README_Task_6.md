**Описание процесса и анализ результатов**

1. Из-за ограничения моих возможностей на ноутбуке, оставила реализацию трех подходов: дискретная фильтрация (Voxel+SOR), непрерывная реконструкция (Poisson) и полигональное моделирование (BPA).
2. Наилучший результат (RMSE = 0.0064) показал метод реконструкции поверхности Пуассона, который эффективно выполнил ресэмплинг (resampling) поверхности, устранив Гауссов шум.
3. Базовый метод воксельной фильтрации (Voxel Downsample) показал стабильный результат (RMSE = 0.0169), но ограничен ошибкой дискретизации сетки.

**Целью данной работы является** восстановление геометрии 3D-объекта после синтетического зашумления. Реализовано: генерация эталона, внесение искажений, очистка, фильтрация, нормализация и сравнительная оценка качества восстановления с использованием метрик RMSE и Chamfer Distance.

**Технические особенности реализации**
1. Ограничения Open3D на ARM64 (Apple M4 Pro). При использовании алгоритма Poisson Surface Reconstruction с глубиной дерева depth=9 происходил критический сбой ядра (Segmentation Fault). Поэтому глубина дерева ограничена значением depth=7, но это дает снижении детализации.
2. Отсутствие Tensor API. В текущей сборке Open3D для macOS (conda-окружение) модуль open3d.t.geometry работает некорректно (ошибка импорта Metric). Расчет метрик Symmetric RMSE и Chamfer Distance реализован вручную с использованием NumPy и Legacy API (compute_point_cloud_distance).
3. Визуализация. Для построения проекций использовался Matplotlib.

# Результаты работы
Классификация 3D-примитивов (5 классов) в условиях зашумления
Классы: Sphere, Cube, Cylinder, Cone, Torus
Модель: SVM (RBF Kernel) + Global FPFH
Итоговая точность (Weighted F1-Score): 1.0000

**Описание процесса и анализ результатов**

1. Генерация данных: синтетический датасет (400 объектов Train, 100 объектов Test). В данные внедрены искажения: Гауссов шум и случайные выбросы (5% точек).
2. Из-за макбука локальные дескрипторы на зашумленных данных работают нестабильно. Поэтому был применен подход Global FPFH: гистограммы локальных геометрических свойств агрегируются (Mean + Std pooling) в единый вектор признаков (размерность 66) для всего объекта.
3. Классификация: 100% точность разделения классов.
4. Визуализация: PCA-проекция пространства признаков.
   
**Целью данной работы является** классификация трехмерных объектов методами классического компьютерного зрения (без использования Deep Learning). Выполнены этапы генерации данных, воксельной предобработки, извлечения признаков FPFH и обучения классификатора.

**Технические особенности реализации**

1. Отказ от детекторов ключевых точек (ISS). Изначально планировалось использование детектора Intrinsic Shape Signatures (ISS). Однако на гладких зашумленных объектах (Сфера, Тор) детектор показывал низкую повторяемость точек. Был осуществлен переход на глобальные дескрипторы, не требующие детекции ключевых точек.
2. Оптимизация под cpu. 
3. Типизация. Для корректной работы sklearn и open3d строгое приведение типов данных (float64 -> float32) при формировании векторов признаков.
